Сборка статического проека на vite с шаблонизатором и препроцессором

##Прототипы
https://kvt-service.tools/

## Использование

1. npm run dev запуск на http://localhost:8080/ в режиме отслеживания

2. npm run build запуск сборки

3. npm run preview запуск в режиме предпросмотра

4. Inspect: http://localhost:8080/__inspect/
